# Headline Study Payment Data Recovery

Please complete this short recovery step using the **same browser and device**
you used for the original headline study. The original study stored your exact
investment choices and generated outcomes in that browser, but those records
were not included in the submitted study file.

This step does not ask you to make any new investment decisions. It only sends
the original locally stored trial records so that the three-trial performance
bonus described in the study can be calculated correctly.
